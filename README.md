## Placeholders:
- Device:
	- `%floodgate_device%`
- Locale:
	- Lowercase: `%floodgate_locale%`
	- Uppercase: `%floodgate_locale_upper%`
- Version
	- `%floodgate_version%`
- Xbox username
	- `%floodgate_username%`
- Xbox xuid
	- `%floodgate_xuid%`
